# FitLife — Gym & Fitness Website

**WPDD202 Assessment 2 (Bootcamp)**
Student: Araf | Student ID: k250741 | Kent Institute Australia

## Project Overview

FitLife is a fully responsive single-page fitness studio website built as a prototype web application for the WPDD202 Bootcamp Assessment. The project demonstrates front-end development skills including semantic HTML5, responsive CSS3, and vanilla JavaScript interactivity.

## Project Structure
## Features

- Responsive Design — Mobile-first layout using CSS Grid and Flexbox
- Semantic HTML5 — Correct use of header, nav, section, article, footer
- CSS Custom Properties — Consistent theming via root variables
- Smooth Scroll Navigation — Fixed sticky nav with anchor-link scrolling
- Mobile Hamburger Menu — JS-powered toggle for small screens
- Scroll Reveal Animations — IntersectionObserver triggers staggered animations
- Contact Form — Client-side validation with submission feedback

## Sections

- Navigation — Fixed header with mobile toggle
- Hero — Full-viewport with headline, CTAs, stats bar
- About — Two-column layout with overlapping visual elements
- Programs — 6-card grid showcasing training programs
- Trainers — 3 coach profiles
- Pricing — 3-tier pricing with featured centre card
- Contact — Address info and inquiry form
- Footer

## Technologies

- HTML5
- CSS3 (Grid, Flexbox, custom properties, animations, media queries)
- Vanilla JavaScript (ES6+, IntersectionObserver API)
- Google Fonts (Bebas Neue + DM Sans)
